
## requirements
- docker
- bun 1.0.29
- git
- python 3.12

## install

1. clone this repo with ``git clone <url>``
2. install dependencies. run the following in the root of the repo.
```
cd impakt-frontend
bun i
cd ../imapkt-backend
pip install -r requirements.txt
cd ..
```
3. run with ``docker compose build && docker compose up``
